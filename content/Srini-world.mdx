Example:

# 2. Embedding Images
You can add images using Markdown syntax or custom components.

Markdown Image:
![Alt text](https://via.placeholder.com/150)

Local Image:

<img src={LocalImage} alt="Example local image" width="300" />



# 3. Links
Use Markdown for links:

Example:

[Go to Google](https://www.google.com)
You can also use custom Next.js <Link> components:

<Link href="/about">Go to About Page</Link>
